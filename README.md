### 8. AI 면접관 Agent (이력서·자소서 기반 면접 Q/A·평가·리포트 자동화)
**Keyword:** LLM, LangGraph, 인터뷰 시뮬레이터  

지원자의 이력서·자기소개서를 기반으로  
**면접 질문 생성 → 답변 평가 → 추가 질문 → 최종 리포트 작성**까지 자동으로 수행하는 AI 면접관 에이전트입니다.

- `InterviewState`를 정의해 이력서 요약, 키워드, 질문 전략, 현재 질문/답변, 평가 로그 등을 상태로 관리
- 이력서 분석 → 질문 생성 → 답변 평가(관련성/구체성/논리성 등) → 진행 여부 결정 → 인터뷰 종료 후 리포트 생성까지 플로우 설계
- LangGraph를 활용해 각 단계를 노드로 나누고, 규칙 기반 로직 + LLM 응답을 조합해 일관된 인터뷰 경험 제공
- 단순 “질문 만들어줘” 수준을 넘어서, **상태 기반 워크플로우로 LLM을 컨트롤**하는 방법을 익힌 프로젝트

### 9. AI 강사 Agent (강의자료 → 수업 설계·퀴즈·피드백 자동화)
**Keyword:** 교육 도메인, LLM, 콘텐츠 생성  

강의자료(PDF/슬라이드/텍스트)를 입력하면  
**수업 구성, 핵심 요약, 퀴즈, 과제, 피드백**까지 자동으로 생성해주는 AI 강사 에이전트입니다.

- 강의자료를 분석해 챕터별 핵심 개념을 추출하고, 난이도에 맞는 학습 목표 및 수업 흐름 설계
- 개념 이해를 점검할 수 있는 객관식/주관식 퀴즈와 간단한 과제를 자동 생성
- 학생 답변 예시에 대해 피드백 문구를 생성하는 프롬프트/에이전트 구조 설계
- “강의자료 → 수업 설계 → 평가 → 피드백” 전체 플로우를 자동화해보며, **교육 도메인에서 LLM을 어떻게 실전 적용할지** 실험한 프로젝트
